# PustakGhar

- PustakGhar is an ecommerce site for books.
- Built using: HTML3 and CSS.
- Components used from [MatrixUI](https://matrixui.netlify.app/) component library.
- Deployed on Netlify.
- Deploy link: https://pustakghar.netlify.app.
